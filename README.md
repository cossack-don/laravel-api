### Задачи
1. Развернуть laravel
2. Интегрировать с БД MySQL
3. Разобраться с CORS, чтобы не мешал. Протестить dir (в ней dir-front, dir-laravel-api)
4. Разобраться с routers
5. Разобраться с resources
